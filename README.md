# Allard Elec - Site Web Professionnel

Site web moderne pour Allard Elec, électricien professionnel en Haute-Savoie.

## Technologies utilisées

- **Next.js 16** - Framework React pour le rendu côté serveur
- **TypeScript** - Typage statique pour un code plus robuste
- **Tailwind CSS** - Framework CSS utility-first pour un design moderne
- **Framer Motion** - Animations fluides et professionnelles
- **Tabler Icons** - Bibliothèque d'icônes moderne et élégante

## Installation

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Construire pour la production
npm run build

# Lancer en production
npm start
```

## Pages du site

1. **Accueil** - Présentation générale des services
2. **Électricité** - Services électriques (constructions neuves, rénovations, dépannages)
3. **Domotique** - Solutions de maison intelligente
4. **Aspiration centralisée** - Systèmes CYCLOVAC
5. **Automatisation Portail/Garage** - Motorisation et sécurisation
6. **Contact** - Coordonnées et formulaire de contact
